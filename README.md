🚀 紅k決策王 (RedK Decision Master)

《紅k決策王》是一款專為台股投資人設計的極速決策輔助工具。它專注於解決一個核心痛點：當昨日漲停收紅K的股票，今日開盤後該如何決策？

透過即時輸入開盤價與昨收價，本工具能立即計算出溢價率，並根據多種技術面規則，提供明確的「續抱」、「減碼」或「清倉」建議。

💡 核心特色

極速決策：手機開啟即可使用，告別複雜看盤軟體，瞬間掌握開盤節奏。

智慧溢價計算：自動對照關鍵價位，協助您在盤中做出果斷行動。

自訂漲跌定義：支援台股傳統紅漲綠跌，也支援國際股市綠漲紅跌切換。

本地資料儲存：您的選股紀錄與偏好設定直接存於手機瀏覽器，完全無須註冊，隱私安全。

深色模式支援：保護您的雙眼，長時間看盤也舒適。

輕量級設計：無需下載繁瑣的 App，網頁即開即用，手機桌機皆可無縫切換。

🛠️ 如何開始使用

直接訪問：請前往 GitHub Pages 連結（請填入您的網址）。

輸入價格：輸入當日的「開盤價」與該股的「昨收價」。

查看策略：工具將自動更新速查表，並根據溢價幅度顯示操作策略建議。

即時決策：

強勢續攻：開盤價大於 5% 關鍵點且支撐有力。

減碼落袋：衝高突破滿足點時的避險建議。

清倉撤退：主力意願不足或開盤低於昨收時的急挫逃命機制。

📱 手機安裝指南 (PWA)

這不是普通的網頁，您可以直接將它「安裝」到手機主畫面上，體驗如同原生 App 的流暢感受：

iPhone (Safari)：點擊分享按鈕 $\rightarrow$ 選擇「加入主畫面」。

Android (Chrome)：點擊右上角三點選單 $\rightarrow$ 選擇「安裝應用程式」。

⚠️ 免責聲明

本工具提供的數據分析與操作建議僅供參考，不構成任何投資建議或買賣保證。股市投資存在風險，請審慎評估個人財務狀況與市場趨勢，使用者自行承擔所有投資決策之後果。

🛠️ 技術堆疊

Tailwind CSS：實現現代化、響應式且輕量的使用者介面。

Font Awesome：視覺化 ICON 圖示。

LocalStorage：實現無需後端、完全本地化的資料存儲。

Web Audio API：輕量化音效回饋，提升操作體驗。
